React 19x component is a function or class that returns a set of instructions to the browser
but they recommended to use function component

componets Rules:
1)componet name should start with uppercase
2)componet should return html/jsx
3)componet should be self closing
4)componet should be reusable

componet types:
1)function component
2)class component

syntax :declaration
function component(){
    return(
        <h1>hello world</h1>
    )
}
syntax :expression
const component = () => {
    return(
        <h1>hello world</h1>
    )
}
- componet function can't be void types
- function currying is where function returns another function
- componet function can't be currying
- componet function can return only one fragment

return(
    <h1>hello world</h1>    //invalid
    <p>hello world</p>
)
return(
    <div>
    <h1>hello world</h1>    //valid
    <p>hello world</p>
    </div>
)
